Recommendations
=======================================

.. _recommendations:

This template provides recommendations for the technical documentation of Fraunhofer products using the static page generator solution `Sphinx`_.

.. hint:: 

   We recommend to learn about `Sphinx`_ and `Technical Writing`_ in general and to review the `Fraunhofer Schreibregeln`_ that are summarized in this recommendations part of the template.
   
The recommendations are clustered into the following:

#. :ref:`Document Recommendations <document-recommendations>`
#. :ref:`Content Recommendations <content-recommendations>`
#. :ref:`Style Recommendations <style-recommendations>`

To help with the transitioning of your existing documents into this template, initial documents are provided according to the :ref:`document recommendations <document-recommendations>`. Refer to the left-hand menu.

Resources
--------------
For information or guidelines that are not addressed in this document, you can consult
the following resources:

- `Fraunhofer Schreibregeln`_ and `Editorial Style Guide`_
- `AP Stylebook`_
- `Microsoft Writing Style Guide`_
- `Google Technical Writing Resources`_
- `Google Style Guide`_
- `Webster's New World Dictionary`_
- `German laws in English`_
- `Title capitalization check`_
